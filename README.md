# Servicio Social - IIMAS UNAM

Institución: Instituto de Investigaciones en Matemáticas Aplicadas y en Sistemas (IIMAS), Universidad Nacional Autónoma de México (UNAM).

Nombre del programa: DESARROLLO DE SISTEMAS INTELIGENTES USANDO INTELIGENCIA ARTIFICIAL.

Proyecto: Comparación de modelos x-vector y ECAPA-TDNN.

Descripción:
En este repositorio se encuentran todos los resúmenes de las investigaciones que realicé para adentrarme al servicio social, así como los códigos de práctica y de experimentación desarrollados durante este proceso. El contenido se enfoca en el estudio y la comparativa técnica entre los modelos x-vector y ECAPA-TDNN, sentando las bases teóricas y prácticas necesarias para el desarrollo del proyecto en el IIMAS.
